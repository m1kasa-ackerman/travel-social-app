Document 2: Design Specification Doc
1. Visual Language (Inspired by Sakib Mahamud)

Style: Clean, high-contrast, card-based layout.

Primary Color: #FF5A5F (Action buttons/Markers).

Secondary Color: #2A2A2A (Typography/Structure).

Corners: 16px border-radius for all image cards and buttons.

2. Key Screen Workflows

Home Tab: Masonry grid of itineraries. On-scroll "Fade-in" animations.

Maps Tab: Full-screen Mapbox integration. Custom custom SVG markers for different categories (Fork/Knife for food, Mountain for treks).

Save Interaction: When "Save" is clicked, a micro-animation moves the icon to the bottom navigation bar.

3. Assets & Components

Iconography: Thin-line SVG icons (Feather Icons or Lucide).

Images: 4:3 aspect ratio for feed consistency.
